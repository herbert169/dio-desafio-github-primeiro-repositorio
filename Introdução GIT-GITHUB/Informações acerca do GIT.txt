				Aula 1.2
		Navegação básica do terminal e instalação
Objetivos da aula:
- Mudar pastas
- Listar pastas
- Criar pastas/arquivos
- Deletar pastas/arquivos

	Comandos a ser utilizados:

*DIR:   Lista pastas
*CD:    Navegação entre as páginas
*CD/:   Pastas especificas
*CLS:   Limpa o terminal
*CD..:  Retorna um nivel na busca
*ECHO:	Printa no terminal o que digitamos
*DEL:   Deleta pasta e arquivo
*MKDIR: Criar pasta
*RMDIR: Apagar repositório
*TAB:   Localiza pasta pelo nome
* > :   Redirecionamento de fluxo
